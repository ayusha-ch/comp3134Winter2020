• What is the definition of server port?
a communication endpoint, where physical and wireless connections are terminated for hardware devices. 
within OS, port identifies a process or type of network service. identified by 16 bit unsigned numbers, aka port number. 
protocols that use port numbers are TCP and UDP.

• What is server port range?
range is from 0 - 65535

• How many bits does number above represent?
16 bits
